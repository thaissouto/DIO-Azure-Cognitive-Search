# DIO-Azure-Cognitive-Search
Repositório sobre Azure Cognitive Search: Utilizando AI Search para Indexação e Consulta de Dados.

No projeto  da DIO, Azure Cognitive Search: Utilizando AI Search para Indexação e Consulta de Dados, aprendi sobre o conceito de mineração de conhecimento e sua importância na extração de informações valiosas a partir de grandes volumes de dados.

Estudei como o Azure Cognitive Search funciona como uma plataforma de mineração de conhecimento alimentada por IA, permitindo uma compreensão mais profunda do conteúdo por meio de recursos como visão computacional, processamento de linguagem natural, e muito mais. Com isso, o processo de indexação torna o conteúdo pesquisável e explorável.

Aprendi também que o enriquecimento de dados com IA torna o conteúdo ainda mais útil para fins de pesquisa. Esse enriquecimento é realizado através de um conjunto de habilidades, como reconhecimento e tradução de textos, além da avaliação de sentimentos. Esses conjuntos de habilidades geram documentos enriquecidos, que são consumidos durante a indexação e, em seguida, enviados ao mecanismo de busca como dados serializados.

No laboratório prático, fomos apresentados a uma situação-problema em que atuamos na área de tecnologia de uma rede nacional de cafeterias. A empresa está recebendo uma grande quantidade de reviews de clientes, e nosso desafio foi utilizar o Azure AI Search para compreender melhor essas avaliações.

Para isso, começamos criando um novo recurso do Azure Cognitive Search. Em seguida, criamos uma conta de armazenamento, ajustamos as configurações de segurança, criamos um container e fizemos o upload das resenhas fornecidas pela professora, que seriam utilizadas para alimentar a estrutura de dados.

Depois, configuramos a fonte de dados e preenchemos os campos necessários, incluindo o tipo de informação que gostaríamos de extrair com base nas reviews. O sistema passou então a retornar dados relevantes diretamente dentro do Azure, permitindo aplicar filtros de localização, como por exemplo, buscando por “Chicago”, onde conseguimos visualizar todas as avaliações relacionadas à cidade.

Aprender sobre todos esses recursos foi uma experiência extremamente enriquecedora. Explorar ferramentas como o Azure Cognitive Search e entender como a inteligência artificial pode ser aplicada para transformar dados em insights valiosos ampliou minha visão sobre o potencial da tecnologia. Desde a mineração de conhecimento até a análise de sentimentos e consultas inteligentes, cada etapa mostrou como essas soluções podem facilitar a tomada de decisões e otimizar processos de forma eficiente.
